## 1.多继承

多继承即一个子类可以有多个父类，它继承了多个父类的特性。

**多继承的基类之间不能有同名的成员**

C++ 类可以从多个类继承成员，语法如下：

```cpp
class <派生类名>:<继承方式1><基类名1>, <继承方式2><基类名2>,…

{
<派生类类体>
};
```

其中，访问修饰符继承方式是**public、protected**或**private**其中的一个，用来修饰每个基类，各个基类之间用逗号分隔，如上所示。
